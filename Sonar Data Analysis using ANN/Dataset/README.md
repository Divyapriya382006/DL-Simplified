# Dataset

## 📌 Description

This project uses the **Sonar Dataset**, which contains sonar signal measurements reflected from either **rocks** or **metal cylinders (mines)**. The objective is to build a binary classification model that can accurately distinguish between the two classes using machine learning techniques.

---

## 📊 Dataset Information

- **Total Samples:** 208
- **Total Features:** 60 numerical attributes
- **Target Classes:**
  - **R** → Rock
  - **M** → Mine

Each feature represents the energy of a sonar signal within a specific frequency band. The dataset contains no missing values and is well-suited for binary classification tasks.
