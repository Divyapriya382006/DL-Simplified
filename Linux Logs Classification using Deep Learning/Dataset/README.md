# Dataset

## Description

This project uses a Linux System Logs dataset containing structured log records generated by Linux operating systems. The dataset is used to train an Artificial Neural Network (ANN) for automatic log event classification.

---

## Dataset Information

- Structured Linux system log records
- Multiple event categories
- Suitable for multi-class classification
- Used for log analysis and event prediction

---

## Dataset Source

The dataset is publicly available on Kaggle.

**Dataset Link:**

https://www.kaggle.com/datasets/ggsri123/linux-logs

---

## Dataset File

```
Linux_2k.log_structured.csv
```
